# 6v-DC-Motor-Water-Pump ⛽
A demonstration of repurposing a dc motor as a water pump

# Parts 🧰
- 6v dc motor
- 9v battery
- Switch
- Copper wire
- 3 plastic lids (2 for the housing, 1 for the impeller)
- Optional - Cable jacket
- Glue gun
- Scissors
- Tape measure
- Pencil
